# Project Overview

## Purpose
Laravel CRUD Generator package that generates complete CRUD modules with Vue 3 + Inertia.js + Shadcn-vue integration.

## Package Name
isaac/laravel-crud-generator

## Description
A Laravel package for generating complete CRUD modules including:
- Laravel backend components (Models, Controllers, Requests, Resources, Migrations)
- Vue 3 frontend components with Inertia.js
- Shadcn-vue UI components
- Full CRUD functionality with best practices

## Technology Stack
- **Backend**: Laravel 12, PHP 8.4|8.3
- **Frontend**: Vue 3, Inertia.js, Shadcn-vue
- **Package Tools**: Spatie Laravel Package Tools
- **Testing**: Pest PHP 4.0
- **Static Analysis**: PHPStan (Larastan 3.0)
- **Code Style**: Laravel Pint

## Package Type
Laravel package (spatie/laravel-package-tools based)
